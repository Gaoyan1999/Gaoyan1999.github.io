                          # Linux 基础

                          # 常用命令

                          目录类

                          • cd: 切换目录命令，以 ”/ “ 表示绝对路径
                          • ./:当前目录
                          • cd ..:退回上一级
                          • ls：展示当前目录下所有文件
                          • pwd: 显示所在目录
                          • mkdir XXX:创建一个目录（-p: 创建多级）
                          • cp复制： cp 某文件 某目录
                          • rm：移除
                          • mv：
                            • 移动文件或者目录 :mv XXX 某文件 /
                            • 重命名 :mv XXX YYY

                          参数

                          • -a: 查看所有文件，包括隐藏
                          • -l: 查看所有文件，以及对应的信息，不包括隐藏。
                          • -f: 忽略不存在的文件，没有警告
                          • -r:递归删除目录
                          • -u:只替换已经更新过的文件

                          # 属性

                          第一个字符代表这个文件是目录、文件或者链接文件等等。

                          • chgrp：更改文件属组
                          • chgrp [-R] 属组名 文件名

                          # 更改权限

                          chmod [-R] xyz 文件或目录

                          Linux 文件的基本权限就有9个，分别是owner、group、others 三种身份的 rwx；

                          r:4 w:2 x:1

                          可读可写可执行 7

                          可读可写 6

                          、、、

                          chmod 777 文件名

                          # 查看

                          ifconfig：查看网络配置

                          网络配置位置： etc/sysconfig/network-scripts

                          • 查看某个文件：

                            普通：cat XXX

                            带行显示：nl XXX

                            上下翻页：less ，/+所查找字符串：查找某个内容。PgUp/PgDn 翻页

                          # 链接

                          硬链接：A---B,若B是A的硬链接，那么它们连个指向同一个文件。意思是一个文件拥有多个路径。

                          软链接：类似快捷方式，源文件没了，快捷方式也访问不了。

                          创建：ln 命令、touch命令

                          创建新文件 ：touch newFile

                          创建硬链接 ：ln f1 f2

                          创建软链接： ln -s newFile newFile f3 （f3 为软链接，f3是蓝色）

                          # 命令模式

                          i/insert:插入

                          x/delete:删除

                          : :切换为底线命令模式

                          # 输入模式

                          用来写文本

                          esc：回到命令模式

                          # 底线命令模式

                          q：退出

                          w：保存

                          wq：保存退出

                          # 账号管理

                          root 用户分配账号

                          添加用户： useradd -选项 用户名

                          -m:自动创建这个用户的主目录 /home/userA

                          删除用户：userdel

                          用户目录一并删除：userdel -r userA

                          修改用户：usermod

                          usermode -d /home/233 userA

                          切换用户：su userA

                          给用户设置密码：passwd userA

                          锁定账户：passwd -l userA

                          # 用户自管理

                          对etc/group 更新

                          创建用户组 ：groupadd groupA

                          指定用户组的id groupadd -g 520 groupA

                          删除用户组： groupdel groupA

                          修改用户组的权限信息和名字： groupmod -g 521 -n groupB groupA

                          切换用户组：newgrp root

                          # 磁盘管理

                          df (列出文件系统整体的磁盘使用量) du(检查磁盘空间使用量)

                          # 进程管理

                          • Linux中每个程序都有自己的一个进程，每一个进程都有一个id号。
                          • 每个进程都有父进程。
                          • 进程可以有两种存在方式：前台、后台。一般服务都是后台运行，程序在前台运行

                          命令

                          • ps：查看当前用户正在执行的各种进程的信息。

                            ps -XX:

                            • -a: 显示当前终端运行的所有进程信息
                            • -u：以用户的信息显示进程
                            • -x：显示后台运行进程的参数
                            ps -aux|eerp mysql
#  | 在Linux中被称为管道符  A|B
#  grep  查找文件中符合条件的字符串
ps -ef|grep mysql
#查看父进程

pstree -pu #查看目录树树
	-p  显示父id
	-u  显示用户组

                          • 杀死进程 kill -9 进程的id

                          # 环境安装

                          • rpm、解压缩、yum在线安装

                          # rpm安装JDK

                          1. 下载 JDK rpm

                          2. 安装java环境：

                            检查当前是否存在java环境，如果有的话就需要卸载

                            rpm -qa|grep jdk # 检查版本信息
rpm -e --nodeps jdk #卸载

# 卸载完毕后即可安装jdk
rpm -ivh rpm包

#安装位置在 /usr/java/jdk1.8.0_251-amd64

#配置环境变量 
vim /etc/profile文件下

JAVA_HOME=/usr/java/jdk1.8.0_251-amd64
CLASSPATH=%JAVA_HOME%/lib;%JAVA_HOME%/jre/lib
PATH=$PATH:$JAVA_HOME/bin;$PATH:$JAVA_HOME/jre/bin
export PATH CLASSPATH JAVA_HOME
